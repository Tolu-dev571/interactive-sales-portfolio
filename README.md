# interactive-sales-portfolio
An interactive sales portfolio website built with HTML CSS and JavaScript for tracking lead generation and calculate sales revenue

Key features:
Lead generation input (lead name, company)
Sales calculate (Number of leads, Conversoin Rate, Price per sale)
Sales and Revenue estimation
Interactive sales dashboard
Real-time data updates

Purpose:
This project domonstrates my ability to combine sales strategy with tecnnology by building a tool that simulates how modern sales team tracks lead and calculate sales per leads with estimated revenue.

Ideal Users
Sales Development Represenatives (SDRs)
Small business teams
Tech sales professionals.
